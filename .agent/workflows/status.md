# STATUS 工作流：项目状态查看

本工作流将显示项目的开发进度、运行状态和关键指标。

## 执行步骤

1. **检查文档状态**

   - ✅/❌ `_Product-Spec.md` 是否存在
   - ✅/❌ `UI-Prompts.md` 是否存在
   - ✅/❌ `_Check-Report.md` 是否存在

2. **检查代码状态**

   - 前端文件数量
   - 后端文件数量
   - 数据库表数量

3. **检查服务运行状态**

   - 前端服务：运行中 / 已停止
   - 后端服务：运行中 / 已停止
   - 数据库：运行中 / 已停止

4. **读取最新检查报告**

   - 如果存在 `_Check-Report.md`，提取进度数据

5. **生成状态报告**

   ```
   📊 项目状态报告

   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   📝 文档状态

   ✅ UI 提示词文档 (UI-Prompts.md)
   ✅ 代码检查报告 (_Check-Report.md)

   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   💻 代码状态
   前端：12 个页面组件，25 个可复用组件
   后端：5 个模块，18 个 API 接口
   数据库：8 张表
   ```
